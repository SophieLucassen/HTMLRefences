How to create a table

Tag Description
Table The first tag to create a table which defines everything within the table.
Table Head Second tag when creating a table and contains the head of the table.
Table Row Third tag that creates a table row.
Table Data Fourth tag that contains the table data.
Table Body Fourth step to create a table, contains the body of the table.
